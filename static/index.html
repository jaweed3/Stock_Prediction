About This Model

This application uses a Long Short-Term Memory (LSTM) neural network to predict Tesla's stock price based on historical data.

The model analyzes patterns in:

  • Historical prices (Open, Close, High, Low)
  • Trading volume
  • Technical indicators (Moving Averages, RSI, MACD, Bollinger Bands)

The prediction is based on the last 60 days of trading data.
